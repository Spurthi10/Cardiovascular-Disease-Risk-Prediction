    About the Model

    CardioGuard is based on peer-reviewed research presented at ICIIBMS 2024, Japan.

    📄 Research Background
    Paper
    "Cardiovascular Disease Risk Prediction Based on Deep Feature Extraction"
    Tao Lin, Mengyao Fan · Shanghai Institute of Technology
    ICIIBMS 2024 · Track 3: Bioinformatics, Bioengineering, Medical Imaging

    Cardiovascular disease (CVD) is the leading cause of death globally, responsible for approximately 17 million deaths per year according to the WHO. Early and accurate risk prediction is critical for preventive intervention. This research proposes a novel deep learning architecture that significantly outperforms traditional machine learning approaches.

    🏗️ Model Architecture

    The model is an end-to-end deep learning pipeline with four main stages:

    1
    Pre-Processing
    Data from UCI and Kaggle (1,024 patient records). Missing values handled, categorical variables encoded, 87 features reduced to 27 via Random Forest + Pearson correlation + Framingham Hazard Scale.
    2
    1D-CNN Feature Extraction
    An improved 1D Convolutional Neural Network captures local temporal patterns in the health data using convolutional layers, pooling layers, and activation functions. The kernel slides along the feature dimension to extract key information.
    3
    Self-Attention Mechanism
    Unlike convolutional layers with fixed receptive fields, the self-attention layer dynamically assigns importance weights to different features, allowing the model to focus on the most predictive factors for each patient's profile.
    4
    Softmax Classification
    The final Softmax layer transforms model output into probability distributions across CVD classes. Trained with cross-entropy loss function for optimal parameter tuning.
